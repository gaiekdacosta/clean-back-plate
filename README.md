## How to use
  #### 1. Download the repository
  ~~~
    git clone https://github.com/gaiekdacosta/cleanbackplate.git 
  ~~~
  #### 2. Install dependencies
  ~~~
  npm install
  ~~~
  #### 3. Create link between local package and project (run as administrator)
  ~~~
    npm link
  ~~~
  #### 4. begin
  ~~~
    initbackend
  ~~~
